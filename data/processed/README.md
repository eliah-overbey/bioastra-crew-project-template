# Processed Data Documentation

This directory documents data products generated by this project.

Unlike the raw data directory, which describes the original datasets used as inputs, this directory describes datasets that were created during the course of the project.

Examples:

* Normalized count matrices
* Differential expression results
* PCA coordinates
* Quality-controlled datasets
* Machine learning outputs
* Aggregated datasets
* Feature tables

Large files should generally not be stored in this repository. Instead, document them here.
